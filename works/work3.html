成都鼓楼清真寺礼拜殿勘测与资产构建

结合摄影测量与游戏引擎技术，实现古建筑的数字化保护与沉浸式展示

Photogrammetry 2025

在省级文物保护项目中，主导古建场景的高精度3D资产采集与前端生成管线。运用摄影测量技术（Photogrammetry）获取古建微观形变数据，为建筑修缮提供科学依据。

Project Overview

本项目为成都鼓楼清真寺礼拜殿木柱倾斜勘测，以省级文保古建筑为对象，运用三维激光扫描、点云建模、构件数字化提取、古建筑形态精准还原等核心技术，完成全建筑毫米级三维数据采集、点云拼接降噪、构件分离拟合与结构形态量化分析，形成高精度三维数字模型、平立剖切片、构件形态数据库等完整数字化成果。

项目过程所学的三维扫描、点云处理、古建筑形制解析、高精度几何重建、构件语义化分类技能，可直接将文物建筑的结构、造型、装饰、空间特征转化为标准化、高精度、可复用的美术资产基底，为游戏资产、虚拟复原、三维展示、文创渲染、数字文博等美术资产制作提供精准数据支撑，实现从实体文物到数字美术资产的高效落地。

Tools Used

Leica RTC360三维激光扫描 单反相机（微观矩阵摄影） Leica Cyclone REGISTER 360 Cyclone 3DR

Project Process

多模态实景采集策略

针对复杂环境，制定“无人机倾斜摄影（屋顶宏观）+激光雷达（精确点云）+单反微距（表面纹理）”的复合采集方案，获取海量底层数据。

海量点云数据处理与降噪

运用Leica Cyclone REGISTER 360对TB级庞大原始数据进行拼接、去重与降噪处理，剔除环境干扰项。

高精数字白模与资产构建

将摄影测量高模导入 Blender，对杂乱三角面进行减面与手动重拓扑优化，独立完成 UV 展开，为 PBR 材质烘焙做好准备，最终输出布线规范、面数合理的高精度 3D 白模，实现物理实体到数字资产的转换。

结构形变数据分析

依托数字模型，对殿内木柱的倾斜状态与损伤程度进行毫米级的精准勘测与量化评估。

Project Outcomes

用户参与度

为文物保护专家及工程师提供了一个可360度自由交互、高精度的数字古建模型，极大提升了研判效率。

用户满意度

提取的形变数据精准可靠，作为科学依据直接指导了后续的针对性物理修缮与保护工程，获得省级文保单位的权威认可。

商业影响力

以高精度数字化勘测构建古建数字底座，赋能数字文博、虚拟展示与文创开发，拓宽文化遗产商业变现与美术资产转化路径。
