独立严肃游戏企划

UI Design Serious game 2025

一款以丹巴嘉绒藏族文化遗产保护为主题的【第一人称模拟经营+RPG】严肃游戏。提取实地调研的非遗基因，让玩家通过修缮与经营藏族碉楼，沉浸式体验文化传承。

Project Overview

这是一款以丹巴嘉绒藏族文化遗产为主题的角色扮演&经营模拟游戏。玩家将化身中路村文化传承入，通过第一人称视角，亲手修复古老经营特色民宿，带领客参与特色民俗活动体验当地风土人情。碉楼、农耕智慧等文化知识点，在文物修复与现代改造游戏融合石砌工艺间寻找平衡。采用清新绘画风格，让玩家在沉浸式体验中，感受嘉绒藏族的建筑智慧与生活美学。游戏最后将通过特色惩罚机制警示，引导玩家思考文化保护的真谛。

Tools Used

Figma Adobe Photoshop XMind 无人机倾斜摄影+全景摄影+光谱检测

Design Process

田野调查与世界观构筑

深入四川中路乡开展沉浸式走访与石匠工艺实践，提取藏羌文化信仰，精准抓取“返乡青年继承破败碉楼”的戏剧冲突，设计“天崩开局”新手引导与多分支叙事脉络。

核心系统与玩法闭环

确立了【场景勘测评估——资源消耗修缮——空间功能转型——游客引流变现】的模拟经营(SIM)核心商业闭环。独立搭建了初版经济系统与UI原型，设计了涵盖基础构件(梯子/窗户)、文化软装(经幡/唐卡)的分类消耗数值，赋予玩家高自由度的养成反馈。

空间关卡设计

发挥专业优势，利用倾斜摄影拆解并还原了传统碉楼的空间组织与生态庭院(高海拔/低海拔植物配比)。

博弈机制与体验平衡

引入“文化保护与现代改建”的数值博弈机制:玩家在经营中需平衡游客的现代居住诉求与传统结构保留，过度现代化将触发隐藏的“惩罚警示机制"，用玩法交互引导玩家反思遗产保护的核心价值。

Project Outcomes

用户参与度

通过“日常经营+节庆传承”交织的叙事线与高自由度的养成系统，成功将枯燥的文化保护课题转化为具备心流体验的沉浸式游玩过程。

用户满意度

游戏概念企划案（Pitch Document）逻辑严密、视觉表现完整，其“惩罚警示机制”的设计获得学界与指导专家的高度认可。

商业影响力

成功打通从“线下文化田野调查”向“线上数字交互IP”转化的全链路研发框架，为地方文旅的数字化传播提供了具备商业落地潜力的游戏化解决方案。
